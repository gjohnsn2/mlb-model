# MLB Model — Notebooks Directory

## Purpose
Jupyter notebooks for exploratory analysis, feature research, and
visualization. These are NOT part of the production pipeline.

## Guidelines
- Name notebooks descriptively: `01_sp_feature_exploration.ipynb`
- Never hardcode paths — use config.py imports
- Document findings in the notebook, then port validated work to
  the pipeline scripts
- Clear outputs before committing (if tracked)
